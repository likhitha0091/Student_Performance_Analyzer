# Student Performance Analyzer

## Overview
A beginner-friendly data analytics project that analyzes student academic performance using Python.

## Features
- Calculates average marks from subject scores
- Assigns grades based on academic rules
- Determines pass/fail status
- Visualizes insights using bar and pie charts

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Output
- CSV file with analyzed results
- Graphs showing average marks and grade distribution

## How to Run
Open the notebook in Google Colab and run all cells sequentially.
